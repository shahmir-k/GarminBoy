This project is tiny lightweight Gameboy emulator for the Garmin Fenix 7x Pro Sapphire Solar smartwatch.

It is meant to reference other tiny and embedded gameboy emulator projects to create a Garmin smartwatch compatible emulator written in Monkey C.



Critical directives:
- Always commit and push on every functional change
- Each commit should have a detailed description
- Large changes should be broken down into organized commits for easy reverting
